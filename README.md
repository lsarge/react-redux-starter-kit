## Unite US Coding Challenge

#### About this project:
I started with some boilerplate from https://github.com/davezuko/react-redux-starter-kit

#### To run this project:
1. install the client dependencies:
```
npm i
```
1. run the the mock API.
3. run the client:
```
npm start
```

#### To run the tests:
```
npm run test
```
